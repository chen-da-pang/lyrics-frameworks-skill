# 歌词填写框架

> 使用说明：
> - 每句后面的 `[X字]` 是字数要求，必须严格遵守
> - `[押主韵]` = 这句句尾必须押本曲选定的主韵（你自己选一个韵，全曲统一）
> - `[散韵]` = 这句句尾不需要押主韵，可以自由换韵
> - `[掺韵]` = 这句故意不押主韵，制造变化，但前后句都押主韵
> - `[导入主韵]` = 这句是从散韵过渡到主韵的最后一句，句尾押主韵
> - Hook 句在全曲中多次重复，必须是同一句话

---

## 全曲结构

- 共 3 轮，11 段，39 句
- 押韵走向：主歌散韵 → 预副歌收韵 → 副歌锁韵 → 尾声连押
- 主韵：自选一个韵，副歌和尾声全部押这个韵

## 第一轮（R1）

### S1 · 第一轮·主歌（4句）

结构任务：建立起点，铺垫状态  
押韵逻辑：自由散韵，不需要押主韵，每句可以换韵

```
L01 [3字]   [散韵]
L02 [11字]  [散韵]
L03 [10字]  [散韵]
L04 [8字]   [散韵]
```

---

### S2 · 第一轮·预副歌（3句）

结构任务：抬升情绪，把听感导向副歌主韵  
押韵逻辑：前两句散韵，最后一句押主韵，完成收韵导入

```
L05 [14字]  [散韵]
L06 [10字]  [散韵]
L07 [5字]   [导入主韵]
```

---

### S3 · 第一轮·副歌一（4句）

结构任务：首次主题宣言，Hook 第一次出现  
押韵逻辑：主韵连押，第3句故意掺韵制造变化，第4句回到主韵

```
L08 [7字]   [押主韵] ← Hook句，全曲重复
L09 [8字]   [押主韵]
L10 [6字]   [掺韵]
L11 [5字]   [押主韵]
```

---

### S4 · 第一轮·副歌二（4句）

结构任务：扩写副歌，Hook 回返  
押韵逻辑：同副歌一，主韵连押，第3句掺韵，第4句回主韵

```
L12 [7字]   [押主韵] ← Hook句（重复L08）
L13 [7字]   [押主韵]
L14 [7字]   [掺韵]
L15 [5字]   [押主韵]
```

## 第二轮（R2）

### S5 · 第二轮·主歌（3句）

结构任务：压缩回返 S1，歌词必须与第一轮主歌高度相似，不能重新创作  
回返规则：S1 有 4 句，S5 压缩为 3 句——把 L01+L02 合并成 L16 一句，L03→L17，L04→L18  
押韵逻辑：同第一轮主歌，自由散韵

```
L16 [14字]  [散韵]  ← 将 L01 和 L02 的内容合并压缩成一句
L17 [10字]  [散韵]  ← 内容与 L03 相同或高度相似
L18 [8字]   [散韵]  ← 内容与 L04 相同或高度相似
```

---

### S6 · 第二轮·预副歌（3句）

结构任务：压缩回返 S2，歌词必须与第一轮预副歌高度相似，不能重新创作  
回返规则：L19 内容与 L05 相同或高度相似，L20 与 L06 相同或高度相似，L21 与 L07 相同或高度相似  
押韵逻辑：同第一轮预副歌，前两句散韵，末句导入主韵

```
L19 [14字]  [散韵]   ← 内容与 L05 相同或高度相似
L20 [10字]  [散韵]   ← 内容与 L06 相同或高度相似
L21 [5字]   [导入主韵] ← 内容与 L07 相同或高度相似
```
