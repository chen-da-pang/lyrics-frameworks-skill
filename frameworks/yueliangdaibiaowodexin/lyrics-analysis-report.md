## 6. 歌词分句映射与真实内韵 (Lyric Phrase Map & True Internal Rhymes)

由于歌词行本身已经是非常短的单句，本框架将每行直接对应为单个 `lyric_phrase`（没有逗号分割的多短语结构），因此 `internal_rhymes[]` 均为空数组，押韵逻辑直接作用在行尾。

---

## 7. 韵律图谱 (Rhyme Map)

* **主原音/主辙 (Main Rhyme)**: **人辰辙 (en/in/un)**
* **辅助/过渡韵**:
  * **S1** 是一韵到底的**人辰辙**（深/分/真/真/心）。
  * **S2/S4** 引入了**破韵设计**（S2: 移[一七]、变[言前]；S4: 想[江阳]、看[言前]），在第三、四句打破原有韵律惯性，制造拉扯感，随后在第五句利用 Hook（心[in]）拉回主韵。
  * **S3 (Bridge)** 使用了人辰/中东的**近韵交织链**（un -> in -> ing -> in，吻/心/情/今），过渡听感平滑。

---

## 8. 非尾韵技法清单 (Non-tail Techniques Inventory)

* **T01 (叠词)**: 作用于 L11 `轻轻` 与 L13 `深深`。首字双字重叠，增强情绪的轻柔度与沉浸感。
* **T02 (动词叠词重复/A一A)**: 作用于 L17 `想一想` 与 L18 `看一看`。形成促使听众/对方采取行动的口语化指令。

## 9. 对举/平行关系映射 (Duiju / Parallelism Map)

* **L01 ↔ L02** (及对应的 L06/L07, L15/L16):
  * **共享字数/句法**: 7字 / 6字；主谓宾结构 + 程度/数量提问。
  * **语义轴**: 设问与回响。
  * **填词约束**: 必须保留这种“提问+追问”的双句对举关系。

* **L03 ↔ L04**:
  * **共享字数/句法**: 5字 / 5字；`我的[名词]也真`。
  * **语义轴**: 平行并列的情感表白。
  * **填词约束**: 保留结构 `我的A也X / 我的B也X`，A、B 为并列词汇，X 为相同押韵字。

* **L08 ↔ L09**:
  * **共享字数/句法**: 5字 / 5字；`我的[名词]不[动词/形容词]`。
  * **语义轴**: 对未来的坚贞承诺。
  * **填词约束**: 保留结构 `我的A不X / 我的B不Y`，做平行否定词对仗。

* **L11 ↔ L13**:
  * **共享字数/句法**: 6字 / 6字；`[AA叠字]的[一个/一段][名词]`。
  * **语义轴**: 记忆中美好的点滴（由动转静，由轻入深）。
  * **填词约束**: 必须使用 `AA的一B` 结构。

* **L17 ↔ L18**:
  * **共享字数/句法**: 5字 / 5字；`你去[动词]一[动词]`。
  * **语义轴**: 行为引导与求证。
  * **填词约束**: 保留 `你去X一X` 结构。
