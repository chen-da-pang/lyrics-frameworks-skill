# 月亮代表我的心 - 邓丽君 | 歌词框架填写模板

> 来源：经典国语流行歌曲《月亮代表我的心》歌词结构。
> 分行口径：采用单行短行分行口径，每行字数为该句歌词的真实字数。
> 主韵：人辰辙 (en/in/un)，本模板不绑定具体韵脚字，仅描述韵路逻辑。

## 全曲结构

- 共 2 轮，4 段，19 句。
- 结构骨架：
  - 第一轮 (R1)：主歌 A (S1) → 主歌 B (S2)
  - 第二轮 (R2)：桥段 (S3) → 主歌 C (S4)
- 句数分配：主歌均含 5 句，桥段含 4 句。
- 韵律走向：以主干韵为主，在主歌后半段（第3、4句）进行破韵打破惯性，并在 Hook 句（第5句）收回主韵。桥段使用平滑近韵链。

## 第一轮 (R1)

### S1 · 主歌 A · 引入与情怀陈述（5句）

结构任务：通过提问与追问切入，陈述情感真实度，并引出核心 Hook 句。  
押韵逻辑：全部押主韵，保证开篇稳定听感。  
技法要求：
- `L01/L02` 形成句式对称的设问与呼应。
- `L03/L04` 形成性质或状态的并列宣告对举，句尾需使用同一个韵尾字。
- `L05` 为核心 Hook 宣言（必须融入歌曲的核心物理/意象载体）。

```text
L01 [7字]  [散韵]     [对举↔L02: 设问引入句，关于抽象距离、时间或深度的提问]
L02 [6字]  [押主韵]   [对举↔L01: 追问呼应句，从另一角度对上述提问进行补充或递进]
L03 [5字]  [押主韵]   [对举↔L04: 主谓并列句法，定中主语 + 副词 + 相同形容词或动作（句尾押主韵）]
L04 [5字]  [押主韵]   [对举↔L03: 主谓并列句法，定中主语 + 副词 + 相同形容词或动作（句尾押主韵）]
L05 [7字]  [押主韵]   [Hook句: 核心情感宣言，结合核心意象做定性宣告]
```

### S2 · 主歌 B · 变体承诺（5句）

结构任务：承接 S1，在第三四句变更为坚贞的忠诚/坚守承诺，保持 Hook 句落点。  
对应关系：`L06 = L01`, `L07 = L02`, `L10 = L05`（字数与文本完全复刻）；而 `L08 ≈ L03`, `L09 ≈ L04` 但词义和押韵改变。  
押韵逻辑：第三四句（L08/L09）进行破韵打破惯性（故意不押主韵），在 Hook（L10）处强力拉回主韵。  
技法要求：
- `L08/L09` 必须形成句法平行的否定式坚守承诺对仗，且尾字均不押主韵。

```text
L06 [7字]  [散韵]     [对举↔L07: 设问引入句] (L01/L06 复刻)
L07 [6字]  [押主韵]   [对举↔L06: 追问呼应句] (L02/L07 复刻)
L08 [5字]  [不押主韵] [对举↔L09: 否定承诺对仗起句，并列名词/代词 + 否定词 + 坚守状态]
L09 [5字]  [不押主韵] [对举↔L08: 否定承诺对仗承句，并列名词/代词 + 否定词 + 坚守状态]
L10 [7字]  [押主韵]   [Hook句: 核心情感宣言] (L05/L10 复刻)
```

## 第二轮 (R2)

### S3 · 桥段 · 记忆与感慨（4句）

结构任务：通过动作和场景回忆，拉开时间/空间跨度，将感情推向高点。  
押韵逻辑：采用 `un -> in -> ing -> in` 近韵链（或对应的其他新主韵的平滑近韵交织链），音感顺滑。  
技法要求：
- `L11/L13` 形成首部叠词修饰的叙事细节对举。
- `L12/L14` 分别作为 L11/L13 的情感反应与时间跨度收口句。

```text
L11 [6字]  [押主韵]   [对举↔L13: 叠词情境起句，[AA叠词]修饰名词或量词+名词，描述过往动作或细节]
L12 [7字]  [押主韵]   [动作带来的情感影响或客观物象的映射，推动情绪]
L13 [6字]  [掺韵]     [对举↔L11: 叠词情境对仗句，[AA叠词]修饰名词，故意不押主韵/押近韵，制造情感对比]
L14 [7字]  [押主韵]   [局部感慨，包含时间、归宿或长久情感的收束]
```

### S4 · 主歌 C · 变体呼唤（5句）

结构任务：回归主歌结构，在第三四句引导对方采取行动或在心理上进行求证。  
对应关系：`L15 = L01`, `L16 = L02`, `L19 = L05`（字数与文本完全复刻）；而 `L17 ≈ L03`, `L18 ≈ L04` 但词义改变。  
押韵逻辑：第三四句（L17/L18）破韵（不押主韵），Hook（L19）拉回主韵。  
技法要求：
- `L17/L18` 必须使用对称的动词重叠式（如“A一A”或类似重叠句式）指引某种感受或求证行动。

```text

L16 [6字]  [押主韵]   [对举↔L15: 追问呼应句] (L02/L16 复刻)
L17 [5字]  [不押主韵] [对举↔L18: 感受/验证指引起句，引导对方通过感官或心智重叠动词进行印证]
L18 [5字]  [不押主韵] [对举↔L17: 感受/验证指引对仗句，引导对方通过感官或心智重叠动词进行印证]
L19 [7字]  [押主韵]   [Hook句: 核心情感宣言] (L05/L19 复刻)
```
