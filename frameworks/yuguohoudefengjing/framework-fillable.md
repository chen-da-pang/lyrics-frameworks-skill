# 歌词填写框架

> 使用说明：
> - 每句后面的 `[X字]` 是字数要求，必须严格遵守
> - `[押主韵]` = 这句句尾必须押本曲选定的主韵（自选一个韵，全曲统一）
> - `[散韵]` = 这句句尾不需要押主韵，可以自由换韵
> - `[掺韵]` = 这句故意不押主韵，制造变化
> - Hook 句在全曲中多次重复，必须是同一句话

---

## 全曲结构

- 共 3 轮，15 段，41 句
- 押韵走向：主歌衣期辙排韵 → 预副歌散韵 → 副歌一/二衣期辙 → 后副歌人辰辙锁韵
- 主韵：衣期辙（i），副歌区和后副歌统一
- Hook-A：副歌一首句（"恨你恨着你 当你带来了分离"），全曲出现 3 次
- Hook-B：副歌二首句（"爱你还爱你 当你带走了回忆"），全曲出现 3 次
- 标题句：后副歌末句（"泪淋湿我的心 等待雨过後的风景"），全曲出现 3 次
- 特殊结构：R3 副歌变体（想你/为你 替换 恨你/爱你）
