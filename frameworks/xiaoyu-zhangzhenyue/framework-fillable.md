# 小宇（张震岳）- 填词模板

原曲主韵：i（衣期辙，主歌主韵）
填词时**不能**以衣期辙（i）作为主歌主韵，请自选其他辙口。

---

## 结构说明

- 两轮结构：R1（主歌一→副歌）+ R2（主歌二→扩展副歌→尾段）
- S1/S3 平行主歌：句式结构几乎镜像，都含比喻对句
- S4 是 S2 的扩展版：前4行≈S2前4行（加"我"字强调），后4行为升级情感新内容
- 长短句交替：10字+6字对（L03/L04, L07/L08, L17/L18）
- 副歌"不管X怎麼樣"排比结构是核心记忆点

## S1 主歌一 R1

> 衣期辙（i）主韵，掺韵丰富。
> 结构：L01起句（8字散韵）→ L02叙述（8字主韵）→ L03长句（10字掺韵）→ L04短句（6字主韵）→ L05-L06比喻对句（8字+8字）→ L07长句（10字）→ L08收束（8字主韵）

```

L03 [10字] [掺韵]  ← 意象描写长句
L04 [6字] [押主韵] ← 意象收束短句（与L03形成10字+6字对）

L06 [8字] [掺韵]   ← 比喻延伸句（"在你的X无处停"句式，与L05形成比喻对）
L07 [10字] [掺韵]  ← 渴望宣言长句
L08 [8字] [押主韵] ← 意象收束句（与L07形成10字+8字对）
```
