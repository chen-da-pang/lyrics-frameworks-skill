## S1 · 主歌 · R1（L01–L04）
# 结构：Hook宣言句×2夹叙述推进句，L01=L03完全重复
# 功能：用同一句话重复两次，中间夹入不同细节，制造"想念→具体渴望"的层进

L01 [11字] [押遥韵锚点词] ← Hook宣言句，全曲遥韵链起点，需与L03/L21/L24/L26押同一韵
L02 [12字] [散韵]         ← 叙述推进句，具体化"想见你"的内容（场景/动作）
L03 [11字] [押遥韵锚点词] ← L01完全重复
L04 [14字] [散韵]         ← 叙述推进句升级，比L02更进一步（情感或行为升级）

## S2 · 副歌 · R1（L05–L10）
# 结构：叠字时间引子开头，6行散韵，言前辙/怀来辙交替
# 功能：从"现在想你"切换到"未来想象"，以对比宣言句收尾
# 注意：L05叠字开头是副歌标志，填词时保留叠字结构

L05 [12字] [散韵，言前辙优先] ← 叠字时间引子句（如"以后以后"→可替换为其他叠字时间词）
L06 [8字]  [散韵]              ← 叙述推进句，短句，承接L05
L07 [10字] [散韵，言前辙优先] ← 对比陈述句（"我的X和你的Y"结构）
L08 [14字] [散韵，怀来辙优先] ← 叙述推进句，含场景动作，双音节词尾承载单押
L09 [6字]  [散韵，言前辙优先] ← 时间引子短句，极短，承接L08
L10 [14字] [散韵，怀来辙优先] ← 对比宣言句收尾（"你是X我是Y"结构），双音节词尾承载单押

## S4 · 桥段 · R3（L17–L20）
# 结构：4行，L18/L20押同一韵（一字韵锁定），L17/L19散韵
# 功能：现实对照，打破副歌的想象，"你不在"重复两次形成情感锤击
# 注意：L18/L20需押同一韵脚字（最好完全相同或极近），制造桥段内锁韵

L17 [10字] [散韵]         ← 时间引子句，建立现实场景
L18 [5字]  [押桥段锁韵]   ← 情感转折句，极短，"可是/但是 + 核心转折词"
L19 [10字] [散韵]         ← 建立场景句，另一个现实场景
L20 [6字]  [押桥段锁韵]   ← L18强化重复，可加语气词（如"喔"）

## S5 · 预副歌 · R3（L21–L23）
# 结构：3行散韵，叠字顿挫句开头，反问句推进，末行为全曲最长行
# 功能：从桥段的情感低点抬升，用反问制造张力，引入S6主歌变体
# 注意：L21叠字顿挫结构（"X X 我们都X"）是预副歌标志

L21 [9字]  [押遥韵锚点词] ← 叠字顿挫句，遥韵链回归（与L01/L03/L24/L26押同韵）
L22 [9字]  [散韵]         ← 反问句，承接L21
L23 [16字] [散韵]         ← 反问推进句，全曲最长行，两个并列分句

# L24 = L01（完全相同），L25 ≈ L02（同字数，细节升级），L26 = L03（完全相同），L27 ≈ L04（同字数，细节升级）
# 功能：全曲以主歌收尾，制造未完成感，情绪停留在"想你"而非"以后"

## 押韵逻辑总结

- **遥韵锚点词**：选定一个主题词作为全曲遥韵锚点，出现在 L01/L03/L21/L24/L26（共5次）
- **桥段锁韵**：L18/L20押同一韵脚（建议完全相同的字或词）
- **其余行**：散韵，无需强制押韵，自然流畅即可
- **副歌**：言前辙（ian/an）与怀来辙（ai）交替出现为佳，但不强制
