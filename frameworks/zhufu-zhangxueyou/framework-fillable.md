# 框架填词模板：祝福（张学友）

原曲主韵：an/ian/uan（言前辙，副歌主韵）
填词时**不能**以言前辙作为副歌主韵，请自选其他辙口。

---

## 结构说明

- 两轮结构：R1（主歌一→预副歌→主歌二→预副歌→副歌）+ R2（副歌回返→扩展→结尾）
- 预副歌出现两次（S2/S4），内容完全相同
- 副歌出现两次（S5/S6），内容完全相同
- 结尾（S8）= S1前6句完整回返 + 压缩预副歌（2句）
- 副歌中三连叠字是核心记忆点，填词时须保留叠字结构

---

## S1 主歌一（第一轮）

> 散韵段，梭波辙倾向但不强制。
> 结构特征：L05/L09为禁止克制句对仗（"不要X不要X"/"莫X莫X"），L11为全段最长句，承担情感转折。

```
L05 [6字] [散韵]
L06 [7字] [散韵]
L07 [7字] [散韵]
L08 [8字] [散韵]
L09 [6字] [散韵]   ← 与L05形成禁止句对仗（句式相同）
L10 [7字] [散韵]
L11 [10字] [散韵]  ← 全段最长句，情感转折爆发点
```

---

## S2 预副歌（第一轮）← Hook句，S4完全重复此处

> 中东辙严韵（ong），两句排韵。
> 功能：从主歌叙事切换到情感宣言，为副歌蓄力。

```
L12 [11字] [押主韵]  ← Hook句，S4 L20一字不差重复
L13 [11字] [押主韵]  ← Hook句，S4 L21一字不差重复
```

---

## S3 主歌二（第一轮）

> 由求辙主韵（ou），中东辙掺韵。
> 结构特征：L14/L17为叠字对仗句（"X许X X许X"/"X难X X难X"），L16为全段最长句（15字），承担感悟爆发。

```
L14 [6字] [押主韵]   ← 叠字对仗句（"几许X几许X"句式）
L15 [7字] [掺韵]
L16 [15字] [押主韵]  ← 全段最长句，感悟爆发点
L17 [6字] [押主韵]   ← 叠字对仗句（与L14形成对仗，"X难X X难X"句式）
L18 [7字] [掺韵]
L19 [10字] [押主韵]
```

---

## S4 预副歌回返（第一轮）

> S2的完全重复，一字不差。

```
L20 = L12（一字不差）
L21 = L13（一字不差）
```

---

## S5 副歌（第一轮）← S6完全重复此处

> 言前辙主韵禁用，自选其他辙口。
> 核心特征：前三句须使用**叠字回环**结构（"X X X X虽然…"/"说X X X X不会…"/"若X X X X就能…"），末句掺韵收束。

```
L22 [10字] [押主韵]  ← Hook句，叠字回环（"X X X X虽然在…"），S6 L26一字不差重复
L23 [10字] [押主韵]  ← Hook句，叠字回环（"说X X X X不会…"），S6 L27一字不差重复
L24 [11字] [押主韵]  ← 叠字回环（"若X X X X就能…"），S6 L28一字不差重复
L25 [11字] [掺韵]    ← 意象收束句，S6 L29一字不差重复
```

---

## S6 副歌回返（第二轮）

> S5的完全重复，一字不差。

```
L26 = L22（一字不差）
L27 = L23（一字不差）
L28 = L24（一字不差）
L29 = L25（一字不差）
```

---

## S7 扩展副歌（第二轮）

> 背景人声重复副歌前两句，主唱唱后两句。

```
（背景）L30 = L22
（背景）L31 = L23
（背景）L32 = L22
（背景）L33 = L23
L34 = L24（一字不差）
L35 = L25（一字不差）
```

---

## S8 结尾（第二轮）

> L36-L41 = S1前6句完整回返（一字不差）
> L42 = L12压缩版（11字→7字，保留核心意象）
> L43 = L13修改版（11字→10字，轻微改动）

```
L36 = L05（一字不差）[6字]
L37 = L06（一字不差）[7字]
L38 = L07（一字不差）[7字]
L39 = L08（一字不差）[8字]
L40 = L09（一字不差）[6字]
L41 = L10（一字不差）[7字]
L42 [7字] [押主韵]   ← L12压缩版，保留预副歌核心意象
L43 [10字] [押主韵]  ← L13修改版，保留陪伴意象
```
