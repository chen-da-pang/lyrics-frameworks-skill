# 《小心願》填词框架

**原曲主韵**：言前辙 uan/yuan（圓/願/段/緣/暖）  
**填词禁用**：不能使用 uan/yuan（言前辙）作为主韵，自选其他韵  
**Hook规则**：全曲所有标注「← Hook句」的位置，Hook句必须完全重复，一字不差  
**字数规则**：严格遵守每句字数，不能多也不能少

---

## S1 第一轮·主歌（L01–L08）

> 结构：4字导入 + 10字叙述×2 + 7字收束，重复两组（圓/完对比）  
> 押韵：散韵为主，偶句押主韵，L05为掺韵（故意不押主韵）

```
L01 [4字] [散韵]
L02 [10字] [押主韵]
L03 [10字] [押主韵]
L04 [7字] [押主韵]
L05 [4字] [掺韵] ← 故意不押主韵，制造转折感
L06 [10字] [押主韵]
L07 [10字] [押主韵]
L08 [7字] [押主韵]
```

**结构逻辑**：L01/L05为同字数导入短句，形成「X是Y」对比（原曲：圓是美滿 / 完是欠缺）。  
L02-L04为第一组叙述，L06-L08为第二组叙述，两组平行。

---

## S2 第一轮·预副歌（L09–L12）

> 结构：9字排比叠字×2 + 7字抬升 + 8字过渡  
> 押韵：四句全押主韵，收紧节奏导入副歌

```
L09 [9字] [押主韵] ← 「誰X X X X不到Y」叠字排比句式
L10 [9字] [押主韵] ← 与L09平行，「誰X X X X快Y」
L11 [7字] [押主韵]
L12 [8字] [导入主韵] ← 过渡句，引入副歌
```

**结构逻辑**：L09/L10为「誰[动词] [叠字] [结果]」排比句式，叠字制造节奏停顿。  
L12为「结尾才是新的开端」式过渡，语义上承接副歌。

---

## S3 第一轮·副歌（L13–L21）

> 结构：Hook-A + 叙述×3 + Hook-B + 叙述×3 + 收束  
> 押韵：掺韵式准排韵，L16/L21为掺韵（故意不押主韵）

```
L13 [8字] [押主韵] ← Hook句A（「[X]的小心願」式）
L14 [10字] [押主韵] ← 「离开一段开一段新一段」式，含重复词
L15 [10字] [押主韵]
L16 [8字] [掺韵] ← 故意不押主韵
L17 [8字] [押主韵] ← Hook句B（「[Y]的小心願」式，与L13同结构但情感反转）← Hook句
L18 [10字] [押主韵]
L19 [8字] [押主韵]
L20 [7字] [押主韵]
L21 [11字] [掺韵] ← 副歌收束句，故意不押主韵
```

**Hook规则**：L13和L17为同一「[X]的小心願」句式，但X不同（情感对比）。  
L17在全曲所有副歌位置完全重复，一字不差。  
L14含「一段」三次重复的节奏技巧（可参考但不强制）。

---

## S4 第二轮·主歌（L22–L25）

> S4 = S1+S2的压缩回返，仅保留预副歌排比叠字结构，省略主歌叙事  
> L22 ≈ L09（同句式），L23 ≈ L10（同句式），L24 ≈ L11，L25为新的情感宣言

```
L22 [9字] [押主韵] ← 「誰X X X X出了Y」叠字排比，平行L09
L23 [9字] [押主韵] ← 「誰X X X X恨裏Y」叠字排比，平行L10
L24 [7字] [押主韵]
L25 [8字] [掺韵] ← 情感宣言句，故意不押主韵，导入副歌
```

---

## S5 第二轮·副歌（L26–L34）

> S5 = S3的完整回返，L26=L13，L27=L14，L28=L15，L29=L16，L30=L17，L31=L18，L32=L19，L33=L20  
> L34 ≈ L21（末句变体，意象从「缺憾」转向「接受圆缺」）

```
L26 [8字] [押主韵] ← Hook句A，与L13完全相同 ← Hook句
L27 [10字] [押主韵] ← 与L14完全相同
L28 [10字] [押主韵] ← 与L15完全相同
L29 [8字] [掺韵] ← 与L16完全相同
L30 [8字] [押主韵] ← Hook句B，与L17完全相同 ← Hook句
L31 [10字] [押主韵] ← 与L18完全相同
L32 [8字] [押主韵] ← 与L19完全相同
L33 [7字] [押主韵] ← 与L20完全相同
L34 [11字] [押主韵] ← L21变体，末句意象升级（原曲：「缺了的圓圈」接受圆缺）
```

**注意**：S5中L26-L33必须与S3中L13-L20完全一致，一字不差。只有L34可以变化。

---

## S6 桥段（L35–L39）

> 结构：哲理长句 + 超长排比句 + 过渡Hook句 + 叙述×2  
> 押韵：散韵为主，情绪从遗憾转向豁达

```
L35 [12字] [押主韵] ← 哲理宣言句，「悲观的豁达点」式情感转折
L36 [24字] [散韵] ← 超长句，三次「差一点的X」排比，口语化
L37 [14字] [押主韵] ← 「Oh [Hook词] [重复句]」过渡呼唤句
L38 [10字] [押主韵]
L39 [15字] [掺韵] ← 哲理收束句
```

**结构逻辑**：L36为全曲最长句，口语化散韵，制造情绪宣泄感。  
L37以「Oh [Hook词]」呼唤引入最终副歌。

---

## S7 结尾（L40–L44）

> S7 = S3/S5的压缩变体，L40=Hook-C变体，L41≈L14变体，L42=L19/L32，L43=L20/L33  
> L44为全曲唯一新收束句，开放式结尾

```
L40 [8字] [押主韵] ← Hook句C变体（「[Z]的小心願」，第三变体）← Hook句
L41 [10字] [押主韵] ← L14变体（「求新一年多一年新一段」式）
L42 [8字] [押主韵] ← 与L19/L32完全相同
L43 [7字] [押主韵] ← 与L20/L33完全相同
L44 [11字] [掺韵] ← 全新收束句，开放式结尾，故意不押主韵
```

**Hook变体规则**：L13/L26为Hook-A，L17/L30为Hook-B，L40为Hook-C。  
三个Hook共享「[X]的小心願」句式，X情感递进（原曲：情人團聚→成為遺憾→仍然懷着）。  
填词时三个Hook的X可以不同，但「的小心願」→替换为你选定的Hook核心词，且全曲一致。

---

## 填词硬约束清单

1. 不能使用 uan/yuan（言前辙）作为主韵，自选其他韵
2. 严格遵守每句字数，不能多也不能少
3. Hook句（L13/L17/L26/L30/L40）在全曲所有标注「← Hook句」的位置完全重复，一字不差
4. [押主韵]的句子句尾必须押选定的主韵
5. [掺韵]的句子故意不押主韵（L05/L16/L21/L25/L29/L34/L39/L44）
6. S5（L26-L33）必须是S3（L13-L20）的完整回返，一字不差
7. S7（L42-L43）必须与L19-L20/L32-L33完全相同
8. 只输出歌词，每句标注行号，不要输出其他内容
